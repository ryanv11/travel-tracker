KEY DECISIONS MADE
================================================================================

  1. Chose Atlas Teal (#0D9488) as primary colour — distinct from generic SaaS
     blue, appropriate for a geographic/travel app.

  2. Chose Inter as the recommended font — superior data-app legibility at small
     sizes, strong tabular figures for dates.

  3. Specified Lucide React (default shadcn/ui icons) to replace all emoji icons.

  4. Specified Sonner for Toast system (shadcn/ui recommended pair).

  5. Recommended max-width reduction from 900/960px → 768px (max-w-3xl)
     for better density, with note to expand to max-w-5xl for future 2-panel layout.
