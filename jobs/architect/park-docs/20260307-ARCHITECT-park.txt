DECISIONS MADE (ADL-01 through ADL-13)
================================================================================

ADL-01  TypeScript throughout. Python rejected due to Electron packaging complexity.
ADL-02  Express.js backend. Fastify rejected (familiarity/ecosystem).
ADL-03  Drizzle ORM. Prisma rejected (SQLite parity). Raw SQL rejected (type safety).
ADL-04  SQLite local → PostgreSQL hosted. Migration = config change only.
ADL-05  REST API. GraphQL rejected (one client, predictable data needs).
ADL-06  Electron for release; localhost-in-browser for beta. Tauri rejected
        (Rust backend incompatible with "same code" constraint).
ADL-07  MapLibre GL JS. Leaflet rejected (raster tiles, poor choropleth).
        Google Maps rejected (cost, API key).
ADL-08  MapTiler free tier for vector tiles.
ADL-09  Natural Earth GeoJSON bundled (countries.json + regions.json).
ADL-10  Nominatim geocoding, offline-tolerant queue-and-resolve (BRD GE-12).
ADL-11  Extension table pattern (Option B) for item type-specific fields.
        Option C (JSON) rejected because rating columns must be indexable (IT-08).
ADL-12  Shading states computed at query time — never stored.
ADL-13  Carry-forward uses dual field: is_carried_forward + carried_from_item_id.
