## Part 2 — ADL-22: Playwright Chromium in Dockerfile

**Decision:** Bake Playwright's Chromium binary into the devcontainer image during
Docker build (before the firewall runs at postStartCommand). This avoids CDN
download at runtime.

### .devcontainer/Dockerfile changes

Two edits:

**Edit 1** — update base image (line 1 or wherever FROM appears):
```dockerfile
FROM node:22    # was: FROM node:20
```

**Edit 2** — add Playwright install block. Insert after the USER root block, before
the final `USER node` line:
```dockerfile
ARG PLAYWRIGHT_VERSION=1.52.0
RUN npx --yes @playwright/test@${PLAYWRIGHT_VERSION} install --with-deps chromium
```

### CRITICAL sync contract (document in a Dockerfile comment above the ARG line)

```dockerfile
# SYNC: PLAYWRIGHT_VERSION ARG must always match @playwright/test in package.json.
# When bumping @playwright/test, update this ARG and rebuild the container.
ARG PLAYWRIGHT_VERSION=1.52.0
```
