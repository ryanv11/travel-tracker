Version: 1.0

================================================================================
PROJECT OBJECTIVE
================================================================================

Build a personal travel tracker desktop application for macOS. The user logs
trips, places, and items (restaurants, hotels, flights, experiences, etc.),
and views their travel history on an interactive world map with country/region
shading based on visit history.

Launch target: Packaged .app (Electron). Beta runs localhost-in-browser.
Future: Hosted web app, iOS app — with no code changes required.

Reference: _project/travel-tracker-BRD.md (v2.3)

KEY DECISIONS (summary — full log in architect/tech/ADL)
================================================================================

  Stack:        TypeScript / Node.js / Express / Drizzle ORM / React / Vite
  Database:     SQLite (Phase 1) → PostgreSQL (Phase 2, config change only)
  Desktop:      Electron (.app release); localhost-in-browser (beta)
  Mapping:      MapLibre GL JS + MapTiler tiles + Natural Earth GeoJSON (bundled)
  Geocoding:    Nominatim (offline-tolerant queue-and-resolve)
  Shading:      Computed at query time — never stored
  Items:        Base table + per-type extension tables
